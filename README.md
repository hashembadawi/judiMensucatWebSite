# Judi Textiles Website

A simple, modern, responsive one-page website for **Judi Textiles** built with **Next.js** and **Tailwind CSS**.

## Features

- Single-page product showcase for textile factory fabrics
- Three language support:
  - Arabic (RTL)
  - English
  - Turkish
- Responsive layout for mobile, tablet, and desktop
- Tailwind CSS utility-first styling
- Clean, maintainable project structure

## Scripts

- `npm run dev` - Start local development server
- `npm run build` - Create production build
- `npm run start` - Run production server

## Deploy on Render

This project includes `render.yaml`.

### Render quick setup

1. Push the project to your Git repository.
2. In Render, create a new **Web Service** from that repository.
3. Render detects `render.yaml` automatically.
4. Deploy.

## Notes

- The site language can be changed from the top language switcher.
- Product data is currently static in `src/app/page.js` and can be connected later to a CMS or API.
